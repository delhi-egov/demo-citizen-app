New building plan application

The process to provide NOC for the plan for a new building has following steps:

  1. You will need to create a new application by click on 'Create Application' button below
  2. On the next screen you will need to fill the Form I
  3. On the following screen you will need to upload the compliance report
  4. On the final screen you will need to click the 'Submit' button to submit the application

Once you create a new application in the first step, it will be stored in our system, and you can resume filling it in even if you dont complete and submit the application in the first go.

The application will be accessible through 'My Applications' link on the Home page.
