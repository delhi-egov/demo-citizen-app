New Building Plan Approval Application

The process to apply for NOC has following steps:
  • Create a new application by clicking Create Application button below
  • Fill the Form I on following screen
  • Upload the Compliance Report on following screen
  • Click Submit button on the final screen
Once an application in created by clicking on Create Application on this screen, it is saved in our system and you can resume filling it if you leave it incomplete by accessing the application from My Applications link on the side bar. The application is only submitted to the department for review once you click Submit on the final screen.
